利用OpenCV2矩形识别+裁剪(拍照识别、文档识别)

自动识别文档矩形区域，并对区域内容纠偏和裁剪，并参照iOS系统`VisonKit.framework`，实现放大镜效果

OpenCV代码调用主要早`WDOpenCVUtils`和`WDOpenCVHelper`中，其中：`WDOpenCVHelper`区域识别和裁剪较为精准，`WDOpenCVUtils`裁剪后有可能会变形
